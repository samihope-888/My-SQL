# My-SQL
Exploration and development of SQL - First project
🚀 SQL & Database Design Project

This project covers the full data lifecycle, from designing a normalized relational database for SkillUp Academy to performing complex multi-table analysis using SQL JOINs and the world database.

🛠️ Skills Highlighted
Database Normalization: Deconstructed a flat-file spreadsheet into linked tables (Students, Courses, Staff, Rotas).
Relational Logic: Implemented Primary and Foreign Keys to establish one-to-many relationships.
Advanced Data Retrieval: Expertly combined data from multiple tables using various JOIN types to answer business questions.
Global Data Analysis: Performed practical SQL queries on the world database to extract insights on countries, cities, and languages.
🔗 Understanding SQL JOINs 
I demonstrated a strong understanding of how to merge datasets based on specific reporting needs
